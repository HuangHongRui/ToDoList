### 为了更了解 npm，要做两件事（可不做）
1. 运行 npm config set loglevel http，让你知道 npm 发的每一个请求
2. 运行 npm config set progress false，关闭那个进度条

### 下载速度更快
1. 运行: npm config set registry https://registry.npm.taobao.org/
2. 恢复: npm config delete registry 即可
